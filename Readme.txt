Proyecto creado para Escanografia SA

Escanografía SA es una empresa dedicada a la toma de imagenes radiológicas como Resonancia magnética, tomografía, radiografía, densitometría, ecografía y pletismografía,
para el diagnostico de enfermedades.

Esta página web ha sido diseñada para dar a conocer los servicios que Escanografía SA ofrece, además de metodos de contacto para agendamiento del mismo y 
para que todos los usuarios tengan mediante la misma web el acceso a los resultados de sus estudios. 

En el proyecto se utilizaron como herramientas los siguientes lenguajes de programación:

-html
-CSS
-php
-Javascript
